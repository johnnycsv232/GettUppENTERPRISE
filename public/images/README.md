# GettUpp ENT Brand Assets

Place your brand assets in this folder:

## Required Images

### Logo
- **`gettup-logo.png`** - The main GettUpp ENT logo (the gold/pink one you shared)
  - Recommended size: 1200x675px (16:9 aspect ratio)
  - Format: PNG with transparency

### Founder Photo
- **`johnny-cage.jpg`** - Johnny Cage's professional photo
  - Recommended size: 800x1000px (4:5 aspect ratio)
  - Format: JPG

### Portfolio Images (in `/portfolio/` folder)
- `last-call.jpg` - Last Call event photo
- `dj-ys.jpg` - DJ YS event photo
- `vip.jpg` - VIP section photo
- `friday.jpg` - Friday event photo
- `opening.jpg` - Grand opening photo
- `artist.jpg` - Artist night photo

Each portfolio image should be:
- Size: 800x1000px (4:5 aspect ratio)
- Format: JPG
- High quality nightlife photography

## Quick Setup

1. Save your logo as `gettup-logo.png`
2. Save Johnny's photo as `johnny-cage.jpg`
3. Add portfolio photos to the `portfolio/` folder

The landing page will display placeholder gradients until these images are added.
